# **App Name**: ArtWorld Central

## Core Features:

- Secure User Authentication: Role-based access control with signup, login, email/OTP verification for public users, talent/members, admins, and super admins.
- Talent Profile Management: Talent/members can manage profiles, upload documents/photos, and view program assignments.
- Admin Talent Management: Admins can approve/reject applications, assign talent to programs, manage photoshoots and track payments.
- Dynamic Registration Forms: Generate program-specific registration forms based on user pathway selection.
- Integrated Payment System: Secure handling of registration, photoshoot, and program fees with mobile money and card payment support.
- Program-Specific Modules: Modular design for CastReal Agency, Miss Pink Ribbon Kenya, and AWI Sports & Arts Club modules with distinct functionalities.
- AI-Powered Content Assistant: AI tool that offers personalized content suggestions to improve profile visibility for talent.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to evoke luxury and exclusivity.
- Background color: Soft ivory (#F8F8FF) for a clean and elegant backdrop.
- Accent color: Dark grey (#4A4A4A) for contrast and a professional feel.
- Body font: 'Inter', a sans-serif font with a neutral and modern feel for all body text.
- Headline font: 'Space Grotesk', a sans-serif font with a geometric and somewhat futuristic vibe, especially suitable for headlines; using 'Inter' for body.
- Custom-designed icons with a minimalist style to represent different programs, actions, and categories.
- Mobile-first responsive design with a clean and intuitive navigation system tailored to each user role.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
